# Pod Lite Assembly Guide

## Tools Required
- Phillips #1 screwdriver
- Anti-static wrist strap
- Torx T6 driver (for AR dock)

## Steps
1. **Mount Compute Module**: Insert RK3588S into base plate
2. **Attach Smart Glass**: Align OLED panel with magnetic guides
3. **Connect Sensors**: Plug MEMS array into labeled ports
4. **Install Ergo Chair**: Bolt to base plate with included hardware
5. **Power On**: Connect solar PSU and press power button

> ⚠️ **Warning**: Always ground yourself before handling compute module
